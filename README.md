MetadataCleaner Pro
MetadataCleaner Pro is a powerful, lightweight privacy tool designed to strip sensitive hidden information from your digital files. Whether you are a journalist, activist, or privacy-conscious user, this tool ensures your files contain only what you want the world to see.

## Key Features
Deep Cleaning: Removes EXIF, IPTC, XMP, and GPS data.

Batch Processing: Clean entire folders of images or documents at once.

Format Support: Works with JPEGs, PNGs, TIFFs, PDFs, and MP4s.

Privacy First: Operates entirely offline. No data is ever uploaded to the cloud.

One-Click Optimization: Automatically reduces file size by removing unnecessary bloat.

## System Requirements
OS: Windows 10/11, macOS 12+, or Linux.

RAM: 512MB minimum.

Storage: 50MB of free space.

## Installation & Usage
To get started with the application, please follow these steps carefully:

Extract the Files: Locate the downloaded archive. Use a tool like 7-Zip or WinRAR to unpack the archive using the provided password in archive .

Launch the App: Open the extracted folder and run the executable file to start the program.

Clean Your Data: Once the interface opens, simply drag and drop your target file into the main window to begin the cleaning process.
